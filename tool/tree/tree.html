🌳 진리의 나무 사용 가이드

⚡ 기본 개념

  • 중앙(8,8)에서 시작해 연결된 칸을 확장하는 노드 배분 시스템이에요
  • 총 포인트 수만큼 노드를 활성화할 수 있어요
  • 반드시 연결된 칸만 활성화 가능해요
  • 중앙 노드는 고정이며 포인트 소모 없이 항상 활성화돼요

🔢 포인트 설정

1
상단 숫자 입력란에 보유 포인트 입력
2
포인트 설정 버튼 클릭 (또는 Enter)
💡 포인트는 최대 289까지 설정 가능해요

✋ 수동 모드

  • 🟡 노란 테두리 칸 → 클릭하면 활성화
  • 🟢 초록 테두리 칸 → 클릭하면 비활성화
  • 연결이 끊기는 칸은 비활성화 불가예요
⚠️ 포인트가 부족하면 새 칸을 활성화할 수 없어요

🤖 자동 탐색 모드

1
상단 탭에서 🤖 자동탐색 클릭
2
목표 선택: 💰골드 / ⚡파워 / 🔬과학 / 🧬(메인)과학종합 / 🛡️던전 / 📉1회용 비용감소
3
🔍 최적 경로 탐색 버튼 클릭
4
결과 확인 후 ✅ 이 경로 적용 클릭
💡 현재 활성화된 노드를 기반으로 추가 최적화해요. 전체 초기화 후 탐색하면 완전히 새로운 경로가 나와요!

📤 Export / Import

  • Export: 현재 배치를 인게임 UID(*TT*...)로 변환 → 게임에 붙여넣기
  • Import: Export로 생성한 UID를 불러와 배치 복원
⚠️ 인게임에서 직접 복사한 UID는 Import가 지원되지 않아요. Export로 생성한 UID만 Import 가능해요.

🔍 효과 검색

상단 검색창에 찾고 싶은 효과를 입력하면 해당 노드가 노란 외곽선으로 강조돼요.

💡 예: "골드" 입력 → 골드 관련 노드 전체 하이라이트

🎨 색상 범례

  • ● 초록 = 활성화된 노드
  • ● 노란 = 활성화 가능한 노드
  • ● 파란 = 자동탐색 결과 노드
  • ★ 보라 별 = 에픽 노드 (복합 효과)
